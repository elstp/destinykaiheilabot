# destinykaiheilabot
开黑啦机器人项目
### 主要功能

- steam arma3服务器查询；


# 使用方法
首先去https://developer.kaiheila.cn/bot/information 创建一个机器人
然后设置机器人连接模式为webhook
复制 Token , Encrypt Key , Verify Token到 插件application.yml里对应的值内
然后复制服务器外网地址 http://IP:端口/webhook?compress=0
填入Callback Url

下载jdk 在此处打开cmd运行 java- jar xxxxxx.jar文件即可使用
